# IndicadoresApp

This project was generated with Angular CLI version 13.3.0.


### Instalar módulos de Node

`npm install`

### Ejecutar proyecto

Ejecutar en la raíz del proyecto el comando `ng serve -o`

### Ejecutar pruebas

Ejecutar el comando `ng test`
