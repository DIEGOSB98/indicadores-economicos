# IndicadoresApp

Proyecto generado con Angular CLI versión 13.3.0 y Node en su versión 14.16.1

### Instalar módulos de Node

`npm install`

### Ejecutar proyecto

Ejecutar en la raíz del proyecto el comando `ng serve -o`

### Ejecutar pruebas

Ejecutar el comando `ng test`
